## ğŸ¬ Step-by-Step: Register as Provider

1. **Find "Manage Roles"** in user menu (top-right)
2. **Toggle "Provider" ON** (the switch turns green)
3. **Click "Save Changes"** at the bottom
4. **Wait for success message** âœ…
5. **You'll be redirected** to provider onboarding
6. **Follow the guide** or skip for later
7. **Done!** Role switcher now appears in header

---

## ğŸ¬ Step-by-Step: Register as Labour

1. **Find "Manage Roles"** in user menu (top-right)
2. **Toggle "Labour" ON** (the switch turns green)
3. **Click "Save Changes"** at the bottom
4. **Fill out the profile form**:
   - Select your skills
   - Enter experience years
   - Set daily rate (required)
   - Set hourly rate (optional)
   - Write a bio
5. **Click "Complete Setup"**
6. **Done!** You're now in labour listings
