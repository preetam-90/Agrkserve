# Admin Booking Detail Page - Design Skeleton

## Overview
Detailed booking view for comprehensive booking information and management.

## Layout Structure

### Header Section
- Back button
- Booking ID
- Status badge
- Action buttons (Edit, Cancel, Refund)

### Booking Summary Card
- Equipment thumbnail + name
- Rental period
- Total amount
- Payment status
- Booking status

### Parties Section (2 columns)
- Renter card (avatar, name, contact)
- Provider card (avatar, name, contact)

### Timeline
- Booking created
- Payment received
- Confirmed
- Equipment picked up
- Equipment returned
- Review submitted
- Completed

### Payment Details
- Amount breakdown
- Payment method
- Transaction ID
- Payment date
- Refund information

### Equipment Details
- Full equipment information
- Images
- Specifications
- Location

### Messages/Communication
- Message thread
- Admin notes
- Add note functionality

### Actions Panel
- Edit booking
- Cancel booking
- Process refund
- Contact parties
- View equipment
- View user profiles
