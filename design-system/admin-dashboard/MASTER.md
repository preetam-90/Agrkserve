# Admin Dashboard - Design Skeleton

## Overview
Main admin dashboard with key metrics, charts, and quick actions for platform management.

## Layout Structure

### Header Section
- Page title: "Admin Dashboard"
- Date range selector
- Quick action buttons (Export, Refresh)

### Stats Grid (4 columns)
1. Total Revenue
   - Icon: DollarSign
   - Value with trend indicator
   - Comparison to previous period
   
2. Active Users
   - Icon: Users
   - Total count with breakdown
   - Growth percentage
   
3. Active Bookings
   - Icon: Calendar
   - Current bookings count
   - Status distribution
   
4. Equipment Listed
   - Icon: Package
   - Total equipment count
   - Category breakdown

### Charts Section (2 columns)
- Revenue Chart (Line/Area)
  - Time-based revenue visualization
  - Multiple data series
  - Interactive tooltips
  
- Bookings Overview (Bar/Column)
  - Booking status distribution
  - Comparison metrics

### Recent Activity Section
- Recent bookings table
- Recent user registrations
- System alerts/notifications
- Quick links to detailed pages

## Color Scheme
- Background: Dark gradient (#0a0a0a to #0f0f0f)
- Cards: Elevated dark (#1a1a1a)
- Accents: Emerald (#10b981), Blue (#3b82f6), Red (#ef4444)
- Text: White/Gray scale

## Interactive Elements
- Hover effects on stat cards
- Animated chart transitions
- Real-time data updates
- Click-through to detailed views
