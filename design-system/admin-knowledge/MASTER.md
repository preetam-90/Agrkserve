# Admin Knowledge Base - Design Skeleton

## Overview
Knowledge base and documentation management for platform content.

## Layout Structure

### Header Section
- Page title: "Knowledge Base"
- Search bar (articles, tags, content)
- Create Article button
- Filter toggle

### Stats Bar (4 metrics)
1. Total Articles
2. Published Articles
3. Draft Articles
4. Views This Month

### Filters Panel
- Status (Published, Draft, Archived)
- Category filter
- Author filter
- Date range
- Tags filter
- Sort options

### Articles List/Grid

#### List View
- Article title
- Category badge
- Author
- Status badge
- Views count
- Last updated
- Actions

#### Grid View
- Article card
- Featured image
- Title
- Excerpt
- Metadata
- Quick actions

### Article Editor (Modal/Page)
- Title input
- Rich text editor
- Category selector
- Tags input
- Featured image upload
- SEO metadata
- Publish/Draft toggle
- Preview button
- Save button

### Categories Management
- Category list
- Add/Edit/Delete categories
- Category hierarchy
- Article count per category

### Analytics
- Most viewed articles
- Search queries
- User engagement
- Content performance
