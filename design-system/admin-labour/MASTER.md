# Admin Labour Management - Design Skeleton

## Overview
Labour/service provider management interface for tracking and managing labour services.

## Layout Structure

### Header Section
- Page title: "Labour Management"
- Search bar (name, service, location)
- Filter toggle
- Add Labour button

### Stats Bar (4 metrics)
1. Total Labour Providers
2. Active Providers
3. Pending Approvals
4. Services Offered

### Filters Panel
- Service type filter
- Status filter (Active, Inactive, Pending)
- Location filter
- Availability filter
- Rating filter
- Sort options

### Labour Providers Table
Columns:
- Avatar + Name
- Service Type
- Location
- Rating
- Availability
- Status badge
- Joined Date
- Actions

Actions:
- View Profile
- Edit Details
- Approve/Reject
- Suspend/Activate
- View Bookings
- Contact Provider

### Provider Detail View
- Profile information
- Services offered
- Availability calendar
- Booking history
- Reviews/ratings
- Documents/certifications
- Performance metrics
