# Admin Payments Debug - Design Skeleton

## Overview
Payment debugging and troubleshooting interface for investigating payment issues.

## Layout Structure

### Header Section
- Page title: "Payment Debugging"
- Search bar (transaction ID, user)
- Refresh button
- Clear filters

### Debug Tools Panel
- Test payment button
- Webhook tester
- Gateway status checker
- Connection diagnostics

### Recent Transactions
- Last 50 transactions
- Status indicators
- Quick filters
- Real-time updates

### Transaction Inspector
- Transaction details
- Gateway request/response
- Webhook logs
- Error messages
- Stack traces

### Gateway Status
- Connection status
- API health
- Rate limit status
- Last successful transaction
- Error rate

### Webhook Logs
- Webhook events
- Payload viewer
- Response status
- Retry attempts
- Timestamp

### Error Analysis
- Common errors
- Error frequency
- Resolution suggestions
- Documentation links

### Test Tools
- Test payment form
- Sandbox mode toggle
- Mock transaction generator
- Response simulator
