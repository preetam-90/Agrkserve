# Admin Settings - Design Skeleton

## Overview
Platform configuration and settings management interface.

## Layout Structure

### Tabs Navigation
- General
- Security
- System
- Notifications
- Integrations
- Advanced

### General Tab
- Site name
- Site description
- Contact email
- Support email
- Platform timezone
- Currency settings
- Language settings
- Maintenance mode toggle

### Security Tab
- Password policies
- Session timeout
- Two-factor authentication
- IP whitelist/blacklist
- API rate limiting
- Security logs

### System Tab
- Database configuration
- Cache settings
- Storage settings
- Backup configuration
- Performance optimization
- Debug mode toggle

### Notifications Tab
- Email templates
- SMS settings
- Push notification config
- Notification preferences
- Alert thresholds

### Integrations Tab
- Payment gateway settings
- Cloudinary configuration
- Third-party APIs
- Webhook management
- OAuth providers

### Advanced Tab
- Feature flags
- Custom CSS/JS
- API keys management
- Developer tools
- System information
