# Admin User Detail Page - Design Skeleton

## Overview
Detailed user profile view for comprehensive user information and management.

## Layout Structure

### Header Section
- Back button
- User avatar (large)
- User name
- User ID
- Role badge
- Status badge
- Action buttons (Edit, Suspend, Delete)

### Quick Stats (4 cards)
1. Total Bookings
2. Total Spent/Earned
3. Account Age
4. Last Active

### Tabs Navigation
- Overview
- Bookings
- Equipment (if provider)
- Reviews
- Activity Log
- Settings

### Overview Tab
- Personal information
- Contact details
- Verification status
- Account statistics
- Recent activity

### Bookings Tab
- Bookings table
- Filter by status
- Booking history
- Revenue/spending chart

### Equipment Tab (Providers)
- Equipment listings
- Performance metrics
- Approval status
- Quick actions

### Reviews Tab
- Reviews given
- Reviews received
- Rating distribution
- Review moderation

### Activity Log Tab
- User actions timeline
- Login history
- Changes made
- IP addresses

### Settings Tab
- Account settings
- Notification preferences
- Privacy settings
- Admin overrides
